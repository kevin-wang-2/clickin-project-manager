## 1. 登录

使用飞书账号登录。点击"飞书登录"后跳转飞书授权页，授权完成后自动返回。

- 首次登录的用户若不在成员列表中，权限为只读（无法编辑）。
- 超级管理员在飞书企业后台中具有"租户管理员"身份，登录时自动识别。

---

## 2. 首页与导航

首页展示所有你有权限访问的剧目，按管理员设定的顺序排列。点击剧目卡片进入该剧目。

页面顶部导航包括：
- **通知**：查看系统发给你的所有通知
- **权限**：查看自己在各剧目中的权限

---

## 3. 剧本与场景

### 场景列表
进入剧目后，左侧菜单 → **场景** 可查看全部场景。场景以树状结构展示，支持嵌套。

### 剧本编辑器
进入剧目后，左侧菜单 → **剧本** 可查看、编辑剧本内容。

- 需要先在顶部选择版本，才能进入编辑状态。
- 只有状态为 **Editing** 的版本可以编辑。
- 支持在文本中使用 `@` mention 成员，`#` mention 剧本页、段落、Cue 或 Asset。

### 导入
支持从外部文件批量导入场景或剧本内容，入口在左侧菜单。

## 4. 版本管理

每个剧目有独立的剧本版本体系，入口在左侧菜单 → **版本**。

### 版本状态
| 状态 | 说明 |
|------|------|
| Editing | 当前可编辑版本，同时只能有一个 |
| Committed | 已提交，内容锁定 |
| Frozen | 重要节点冻结，内容锁定 |
| Archived | 已归档，默认隐藏 |

### 创建新版本
- 基于任意版本创建新版本，内容继承自该版本。
- 若基于 Editing 版本创建，原版本自动变为 Committed，新版本为 Editing。

### 回退
回退不会删除历史，而是：先将当前 Editing 提交为 Committed，再以目标祖先版本的内容新建一个 Editing 版本。

## 5. Cue List

左侧菜单 → **Cue List** 查看、管理所有 Cue Sheet。

- 每个 Cue Sheet 有唯一的全大写简称（如 `SQ`、`LX`）。
- Cue 与剧本块关联，支持在剧本中跳转到对应 Cue。
- Cue 内容可以 mention 剧本位置或 Asset。

---

## 6. 构作（Dramaturgy）

左侧菜单 → **构作** 提供场景级别的可视化视图，用于排布剧本结构、标注角色出现情况等。

---

## 7. 日程与事件

左侧菜单 → **日程** 查看所有排练和演出事件。

### 事件详情
每个事件包含：
- **日程表**：当天的时间安排，可添加分项
- **Call Times**：各部门/人员的到场时间
- **技术需求**：灯光、音响、舞美等技术 rider
- **报告**：排练/演出结束后的总结记录

### Call Sheet
每个事件可生成 Call Sheet（公告单），可通过链接分享给外部人员查看，无需登录。

### 关注事件
点击事件详情右上角的"关注"，可订阅该事件的通知推送。

## 8. Asset 资产管理

左侧菜单 → **Asset** 管理剧目相关的所有文件资产。

### 上传文件
点击右上角"上传"，支持三种方式：
- **点击选择**：点击上传区域选择文件
- **拖拽**：将文件直接拖入上传区域
- **粘贴**：在上传面板打开时，从 Finder 复制文件后按 `Cmd+V`

单次只能上传一个文件。图片上传后自动生成缩略图。

**文件来源说明：**
- 默认上传到 Cloudflare R2（所有格式均支持）。
- 也可以填写飞书 Wiki 节点链接，系统会存储链接而非文件本身。

### 文件类型
上传时选择文件用途类型：图纸、平面图、Demo、排练视频、Reference、素材、片段、QLab、乐谱、录音。

### 在线预览
支持在浏览器内直接预览以下格式：
- **图片**：直接展示
- **视频**：自建播放器（见下方快捷键）
- **音频**：波形播放器
- **PDF**：内嵌 PDF 阅读器

其他格式显示下载按钮。飞书链接直接跳转飞书。

### 播放器快捷键（音频 / 视频）
| 按键 | 操作 |
|------|------|
| `空格` | 播放 / 暂停 |
| `Shift` + `.` | 加速一档 |
| `Shift` + `,` | 减速一档 |

支持倍速：0.5× / 0.75× / 1× / 1.25× / 1.5× / 2× / 3×。  
点击播放器右侧的速度标签（如 `1×`）也可弹出选择菜单。

### 挂载点
Asset 可以挂载到剧目的特定位置（如某个场景、某个 Cue），方便在对应位置直接找到相关文件。挂载关系为多对多，一个 Asset 可以挂载到多个位置。

### 版本绑定
上传时可以选择将 Asset 绑定到当前剧本版本，或设为跨版本通用。

## 9. 角色与联系人

- **角色**：左侧菜单 → **角色**，管理剧目中的所有角色及其属性。
- **联系人**：左侧菜单 → **联系人**，管理剧组成员联系方式。

---

## 10. 通知

顶部导航 → **通知** 查看所有系统通知，包括：
- 事件日程提醒（每日 call / 每周 call）
- 技术需求状态变更
- 被 @ mention 的消息

可在通知设置页面（右上角图标）配置各类通知的推送方式。

---

## 11. 权限查看

顶部导航 → **权限** 查看自己当前的权限状态：

- 是否为超级管理员
- 在每个剧目中拥有的角色
- 各权限项的实际状态（含手动覆盖的条目）
